# Scope options (targeted dogfooding)

When the user does not specify a page or feature, present these options so they can choose a focused scope.

## Options to show

| Option | What to explore |
|--------|------------------|
| **Homepage / welcome** | Landing after login: hero, CTAs, navigation |
| **Product catalog** | Browse products, categories, search, filters |
| **Checkout flow** | CRUD shopping cart, order submission |
| **Surprise me** | You are free to explore however you want |
